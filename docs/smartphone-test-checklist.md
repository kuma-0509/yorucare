# ヨルケア Phase 1 — スマホ実機テストチェックリスト

**目的**: 公開 URL 上で、毎日 1〜2 分の記録体験がスマホで使えるかを検証する。  
**対象**: Phase 1 MVP（ログインなし・localStorage のみ）  
**想定時間**: 最短ルート約 5 分 / 全項目約 20〜30 分

---

## テスト前の準備

| # | 項目 | 完了 |
|---|------|:----:|
| 0-1 | テスト端末のブラウザを決める（推奨: **Safari（iPhone）** または **Chrome（Android）**） | ☐ |
| 0-2 | 公開 URL を開く（例: Vercel の本番 / プレビュー URL） | ☐ |
| 0-3 | **ホーム画面に追加**（任意だが推奨）— 以降のアプリ感・全画面表示の確認用 | ☐ |
| 0-4 | テスト日時・端末・OS・ブラウザをメモ（不具合報告用） | ☐ |
| 0-5 | 初回テストなら **シークレット / プライベートではない通常タブ** を使う（localStorage 検証のため） | ☐ |

**注意（Phase 1 の前提）**

- データは **この端末のこのブラウザだけ** に保存される。
- 別端末・別ブラウザ・キャッシュ削除でデータは共有されない / 消える可能性がある。

## Phase 1 で「不具合ではない」こと

| 現象 | 理由 |
|------|------|
| ログイン画面がない | Phase 1 スコープ外 |
| 3 日より前の記録を編集できない | 今日・昨日のみ編集可 |
| 別端末でデータが見えない | localStorage のみ |
| ふりかえりで週次サマリーが出ない | 準備中（Phase 2） |
| AI・LINE・エクスポートなし | Phase 2 以降 |
